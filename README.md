# Kind's Crystal Optimizer

High-performance client-side crystal optimizer for Fabric.

## Minecraft Versions

- 1.21.1 to 1.21.11

## Build

```bash
./gradlew build -x test
```

Output jar:

`build/libs/Kind's Crystal Optimizer-1.0.0.jar`
